##  PAQUETES INSTALADOS - EJECUTAR ESTE PROGRAMA
### Virtual environment
Este modelo será configurado usando pipenv.:
```sh
pipenv install --python 3.9.2
```
```sh
pipenv shell
```
### Dependencias
Instalar estas dependencias:

```sh
pipenv install scikit-criteria numpi pandas
```
